# ATS Feature & Capabilities Matrix

This matrix charts every active functional capability in the Applicant Tracking System (ATS), defining roles, routers, database schemas, and multi-tenant security verification statuses.

| Feature Module | Purpose / Description | Primary Role | Frontend Page / View | Backend Route Endpoint | Database Tables | Status | Security Scoping Verified |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **Landing & Core Shell** | Public portal, hero metrics overview, and quick demo credentials display panel. | All / Public | `Home.jsx` | `/` (Static) | None | **COMPLETE** | Yes (Publicly readable) |
| **Authentication & Signup** | Secure login, password hashing, and user registration with automatic organization creation. | All | `login.jsx`, `Register.jsx` | `/api/auth/register`, `/api/auth/login` | `users`, `organizations`, `memberships` | **COMPLETE** | Yes (Password hashed) |
| **Recruiter Dashboard** | Live stats indicators, recruitment pipeline charts, and lists of recent candidate actions. | Admin, Recruiter | `Dashboard.jsx` | `/api/dashboard/stats`, `/api/dashboard/recent-activities` | `job_descriptions`, `applications`, `activities` | **COMPLETE** | Yes (Tenant organization filtered) |
| **Job Management** | Create job requisitions, edit JDs, track salary ranges, locations, and skills requirements. | Recruiter, HR | `Jobs.jsx`, `ManageJD.jsx` | `/api/job-descriptions` (POST, GET, PUT, DELETE) | `job_descriptions` | **COMPLETE** | Yes (Ownership validated via `verifyOwnership`) |
| **Candidate Application** | Portal for public applicants to view postings, fill details, and upload resume files. | Candidate | `AvailableJobs.jsx`, `ApplyJob.jsx` | `/api/applications` (POST) | `applications`, `candidates` | **COMPLETE** | Yes (Uploaded to secure storage) |
| **AI Resume Screening** | Automatically analyze resumes against jobs, calculating scorecards, strengths, and missing tags. | Recruiter, HR | `AICandidates.jsx` | `/api/ai/screen/:appId` | `applications` | **COMPLETE** | Yes (Scoping checked before screen) |
| **Candidate Ranking** | Directly sort and compare candidates based on computed AI match score metrics. | Recruiter, HR | `TopCandidates.jsx` | `/api/applications/top-candidates` | `applications` | **COMPLETE** | Yes (Tenant organization filtered) |
| **Kanban Pipeline** | Visual board tracking applicant stages (Applied -> Screening -> Shortlisted -> Interview -> Hired/Rejected) with drag & drop. | Recruiter, HR | `Applications.jsx` | `/api/applications/stage` (PUT) | `applications`, `activities` | **COMPLETE** | Yes (Tenant ownership checked on update) |
| **Interview Management** | Schedule interview slots, specify times, select rounds, write feedbacks, and submit scorecard ratings. | Recruiter, Interviewer | `Interviews.jsx` | `/api/interviews` (POST, GET, PUT), `/api/interviews/email/:email` | `interviews`, `users` | **COMPLETE** | Yes (Recruiters restricted to Org interviews) |
| **Collaboration & Comments** | Mention team members, write notes, and post evaluation reviews on candidate application screens. | Recruiter, Admin | `TeamComments.jsx` | `/api/comments` (POST, GET) | `comments`, `memberships` | **COMPLETE** | Yes (Tenant membership verified) |
| **Recruitment Analytics** | Render bar charts of applicant stages distributions, and match scores spreads. | Admin, Recruiter | `Analytics.jsx` | `/api/dashboard/analytics` | `applications`, `job_descriptions` | **COMPLETE** | Yes (Tenant organization filtered) |
| **Audit Activity Log** | View append-only logs of security actions, database edits, and logins. | Admin | `Settings.jsx` (Audit Log tab) | `/api/settings/audit-logs` | `audit_logs` | **COMPLETE** | Yes (Restricted strictly to Admin role) |
| **Tenant Administration** | Customize organization details, website configurations, default interview durations, and default ranges. | Admin | `Settings.jsx` (Organization tab) | `/api/settings/organization` (GET, PUT) | `organizations` | **COMPLETE** | Yes (Tenant organization isolated) |
